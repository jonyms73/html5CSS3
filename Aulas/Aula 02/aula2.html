Aula 2 - Meta tags, textos e listas

Meta tags

Meta tags são linhas de código que descrevem o conteúdo do site (SEO)

viewport

A meta tag description instrui o navegador como controlar o tamanho e o dimensionamento da página

charset

A meta tag charset é usada para indicar o padrão de codificação de caracteres

description

A meta tag description descreve a página para os buscadores (SEO) - máximo 156 caracteres
